ProB 2.0 for Rodin Updatesite

From ProB Documentation

About this Updatesite

This site contains the experimental ProB 2.0 plug-in for Rodin. Please install it using the Rodin installer for plug-ins. You need to add the URL 

http://nightly.cobra.cs.uni-duesseldorf.de/experimental/updatesite/
to the list of update sites.

Warning

Please note that this site is automatically generated from the lastest automatic integration build. While we try our best to keep the builds wroking it might not lways be the case that you can install or use the tool. Please do not use this tool on the only copy of your production models!
