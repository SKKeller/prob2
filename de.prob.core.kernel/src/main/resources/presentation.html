Data Visualization in ProB

by Joy Clark

Motivation

  • Benefits of visualization
  • Lack of data visualizations in ProB 2.0
  • Move away from GraphViz as visualization engine
ProB is a tool that uses consistency checking and animation to verify formal models. We wanted to implement a way to dynamically generate visualizations based on the data that is produced during verification. This would make it easier for the user to analyze and understand the results that are created from ProB. ProB 2.0 is the Java API that is currently in development for the ProB tool. There are several visualizations currently available in the Tcl/Tk version of ProB that are were not yet available in the ProB 2.0 application.
