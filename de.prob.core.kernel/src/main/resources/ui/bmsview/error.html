How to start a visualisation

To start a visualisation use the following URL:

http://localhost:[port]/bms/?template=[path to template folder]/template.html&lang=csp

Example: http://localhost:8080/bms/?template=/home/user/mytemplates/mycspvis/cspvis.html&lang=csp

How to start the graphical editor / How to create a new visualisation template

http://localhost:[port]/bms/?template=[path to template folder]/template.html&lang=csp&editor

Example: http://localhost:8080/bms/?template=/home/user/mytemplates/mycspvis/cspvis.html&lang=csp&editor

Please note: If the entered visualisation template does not exist it will be created when starting the graphical editor.
